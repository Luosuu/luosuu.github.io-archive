how to use Qemu on macOS

install

brew install qemu

usage

-redir 的命令在mac上无效，使用了

qemu-system-i386 -hda zerowine.img -boot c -m 1024 -net user,hostfwd=tcp::8000-:8000 -net user,hostfwd=tcp::2022-:22 -net user,hostfwd=tcp::8001-:8001 -snapshot -net nic

成功启动了zerowine的虚拟机。
