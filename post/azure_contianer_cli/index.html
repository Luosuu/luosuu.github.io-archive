Azure CLI to Deploy Containers

  • May 13, 2020
  • Software  

最近因为一些原因在折腾Azure，毕竟不能浪费微软爸爸发的每月150刀额度。

创建资源组

az group create \
	--name <Resourse Group Name> \
	--location eastus

运行容器

az container create \
	--resource-group <Resourse Group Name> \
	--name <Container Tag Name> \
	--image <Your IMG> \
	--ports 8388 \
	--location eastus \
	--ip-address public \
	--restart-policy Always \
	--environment-variables \
		PASSWAORD=<YOUR PWD> \
		METHOD=chacha20-ietf-poly1305

检查状态

az container show \
	--resource-group <Resourse Group Name> \
	--name <Container Tag Name>

在容器中执行命令

az container exec \
  --resource-group learn-deploy-aci-rg \
  --name mycontainer \
  --exec-command /bin/sh
