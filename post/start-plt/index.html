准备工作

我推荐开始之前先看看上面的链接里第二个和最后一个，了解一下大概的情况以及这个领域关心的问题。

然后这里搬运一下macOS如何安装MIT-scheme，因为SICP的代码都是基于MIT-scheme的，不过只要是符合IEEE-1990的scheme标准的lisp方言，都可以运行SICP第二版的代码（第二版前言述）。

首先下载MIT-Scheme的 .dmg文件。

64-bit

由于macOS Catalina只支持64位的软件了，所以我推荐下载64位的。下面的是32位的下载链接:

32-bit

下载完成后，打开dmg文件把app拖进Application文件夹。

64位请打开terminal运行

sudo ln -s /Applications/MIT\:GNU\ Scheme.app/Contents/Resources /usr/local/lib/mit-scheme-x86-64

一般会要求你输入密码

然后运行

sudo ln -s /usr/local/lib/mit-scheme-x86-64/mit-scheme /usr/bin/scheme

如果不行，把 /usr/bin/scheme换成/usr/local/bin/scheme

现在你可以在terminal里直接输入 scheme 启动了，启动后输入(+ 1 1)看看是不是运行正常。

上述方法搬自JACKSON’S BLOG

之后你就可以开始草书了。
